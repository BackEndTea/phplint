phplint
=======

`phplint` is a tool that can speed up linting of php files by running several lint processes at once.


# Installation

```shell
$ composer require overtrue/phplint
```

# Usage

```shell
$ ./vendor/bin/phplint ./ --exclude=vendor
```

more:

```shell
$ ./vendor/bin/phplint --help
```
